inferring shorthand argument names from context

When writing an inline closure expression, shorthand argument names can be inferred through the simple inclusion of an operator.

For further info:

  • https://docs.swift.org/swift-book/LanguageGuide/Closures.html

Sample Code: 


reversedNames = names.sorted(by: >)
