Identifiable

A protocol used to identify a type that can be uniquely identified by an id.

For further info:

  • https://developer.apple.com/documentation/swift/identifiable
  • Link 2: https://nshipster.com/identifiable/

Sample Code: 


protocol Identifiable {
    associatedtype ID: Hashable
    var id: ID { get }
}
