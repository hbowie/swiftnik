repeat

The repeat-while loop always performs a single pass through the loop before evaluating the loop's condition.

Also see while.

For further info:

  • https://docs.swift.org/swift-book/LanguageGuide/ControlFlow.html

Sample Code: 


repeat {
	// statements go here
} while // condition goes here
