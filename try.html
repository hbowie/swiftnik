try

Precedes an expression that might throw an error. Generally combined with do and catch.

For further info:

  • https://docs.swift.org/swift-book/LanguageGuide/ErrorHandling.html

Sample Code: 


do {
    try expression
    statements
} catch pattern 1 {
    statements
} catch pattern 2 where condition {
    statements
} catch {
    statements
}
