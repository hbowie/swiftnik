Cocoa

Cocoa is Apple's native object-oriented application programming interface (API) for its desktop operating system macOS.

When importing Cocoa into a Swift application, the following frameworks are implicitly imported.

  • AppKit
  • CoreData
  • Foundation

For further info:

  • https://en.wikipedia.org/wiki/Cocoa_(API)

Sample Code: 


import Cocoa
