init

An initializer is a special method that can be called to create a new instance of a particular type. Note that the keyword init replaces both the func keyword and the function name, when compared to other method calls.

For further info:

  • https://docs.swift.org/swift-book/LanguageGuide/Initialization.html

Sample Code: 


init() {
    // perform some initialization here
}
