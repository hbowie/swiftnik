Title:  catch

Tags:   Swift

Link:   https://docs.swift.org/swift-book/LanguageGuide/ErrorHandling.html

Code: 

do {
    try expression
    statements
} catch pattern 1 {
    statements
} catch pattern 2 where condition {
    statements
} catch {
    statements
}


Body: 

Indicates the sorts of errors you'd like to catch. Generally combined with [[do]] and [[try]].
